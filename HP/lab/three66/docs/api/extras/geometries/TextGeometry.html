[name]

This object creates an 3D object of text as a single object.

Constructor

[name]([page:String text], [page:Object parameters])

text 窶・The text that needs to be shown.
parameters 窶・Object that can contain the following parameters.
  • size 窶・Float. Size of the text.
  • height 窶・Float. Thickness to extrude text. Default is 50.
  • curveSegments 窶・Integer. Number of points on the curves.
  • font 窶・String. Font name.
  • weight 窶・String. Font weight (normal, bold).
  • style 窶・ String. Font style (normal, italics).
  • bevelEnabled 窶・Boolean. Turn on bevel. Default is False.
  • bevelThickness 窶・Float. How deep into text bevel goes. Default is 10.
  • bevelSize 窶・Float. How far from text outline is bevel. Default is 8.
