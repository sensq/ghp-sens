[name]

Creates extruded geometry from a path shape

Constructor

[name]([page:todo shapes], [page:Object options])

shapes 窶・Shape or an array of shapes.
options 窶・Object that can contain the following parameters.
  • curveSegments 窶・ int. number of points on the curves
  • steps 窶・ int. number of points used for subdividing segements of extrude spline
  • amount 窶・ int. Depth to extrude the shape
  • bevelEnabled 窶・ bool. turn on bevel
  • bevelThickness 窶・ float. how deep into the original shape bevel goes
  • bevelSize 窶・ float. how far from shape outline is bevel
  • bevelSegments 窶・ int. number of bevel layers
  • extrudePath 窶・ THREE.CurvePath. 3d spline path to extrude shape along. (creates Frames if (frames aren't defined)
  • frames 窶・ THREE.TubeGeometry.FrenetFrames. containing arrays of tangents, normals, binormals
  • material 窶・ int. material index for front and back faces
  • extrudeMaterial 窶・ int. material index for extrusion and beveled faces
  • uvGenerator 窶・ Object. object that provides UV generator functions
